# Thera Bank Marketing Intelligence Dashboard

A Streamlit dashboard that turns the bank personal loan modelling dataset into an executive-facing marketing intelligence tool.

## Created by
Powell A. Ndlovu

## Features
- Executive overview with KPIs
- Customer targeting explorer
- Model validation command center
- ROI simulator with threshold controls
- Lift and cumulative gain strategy views
- Prediction App

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Files
- `app.py` – main Streamlit application
- `Bank_Personal_Loan_Modelling.csv` – source dataset
- `bank_loan_outputs_portfolio/` – precomputed model outputs and charts
